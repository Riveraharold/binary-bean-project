# Binary Bean Project ☕

Final Project for Web Technology - Modern Web Interface Development.

## 📌 Project Overview
Ang Binary Bean ay isang coffee shop website project na ginawa para sa aming finals. Layunin nitong magbigay ng simple, malinis, at madaling gamitin na platform para sa mga mahilig sa kape. Ipinapakita ng website ang iba’t ibang produkto, menu, at impormasyon tungkol sa coffee shop gamit ang modern at responsive na design.
Ginamitan ito ng HTML, CSS, at Bootstrap upang masigurong maayos ang itsura at gumagana ito sa iba’t ibang devices tulad ng cellphone at computer. Sa kabuuan, ang proyekto ay nagpapakita ng aming kakayahan sa web design at pagbuo ng user-friendly na website.

## 🚀 Technologies Used
- HTML5:
Ginamit para buuin ang structure ng website tulad ng headings, paragraphs, images, at sections. Ito ang nagsisilbing backbone ng buong web page.

- CSS3 / Tailwind CSS:
Ginamit para sa design at styling ng website tulad ng kulay, layout, spacing, at responsiveness. Ang Tailwind CSS ay nakatulong para mapabilis ang paggawa ng modern at malinis na UI.

- JavaScript:
Ginamit upang magdagdag ng interactivity sa website tulad ng button functions, animations, at dynamic na behavior ng mga 
elements.

- Figma:
Ginamit bilang design tool para i-plan at i-layout ang itsura ng website bago ito i-code. Dito ginawa ang UI design at prototype ng Binary Bean.

- Netlify / Vercel:
Ginamit para i-deploy o i-publish ang website online upang ma-access ito ng ibang users gamit ang internet.

## 🛠️ Features
- **Responsive Design**: Gumagana sa mobile, tablet, at desktop.
- **Accessible UI**: Sumusunod sa WCAG contrast standards.
- **Interactive Elements**: May working image slider at contact form validation.
